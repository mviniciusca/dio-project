Guia de Elementos HTML Básicos

O que é um Parágrafo (<p>)?

O elemento <p> define um parágrafo. É um dos elementos mais básicos do HTML e é usado para agrupar texto relacionado. Cada parágrafo cria automaticamente uma quebra de linha antes e depois dele.

Este é outro parágrafo que demonstra como múltiplos parágrafos ficam separados visualmente.

Lista Não Ordenada (<ul>)

Uma lista não ordenada cria uma lista com marcadores (bullets):

  • Primeiro item da lista não ordenada
  • Segundo item - os marcadores são iguais
  • Terceiro item
  • Quarto item com texto em itálico

Lista Ordenada (<ol>)

Uma lista ordenada cria uma lista numerada automaticamente:

  1. Primeiro item da lista ordenada
  2. Segundo item - numerado automaticamente
  3. Terceiro item
  4. Quarto item com texto em negrito

Blockquote (<blockquote>)

O elemento blockquote é usado para citações longas ou trechos destacados:

"A educação é a arma mais poderosa que você pode usar para mudar o mundo. O conhecimento em HTML é o primeiro passo para criar páginas web incríveis e acessíveis para todos."

- Exemplo de citação em blockquote

Outros Elementos de Texto

Aqui estão alguns elementos importantes para formatação de texto:

  • <strong> - para texto importante/negrito
  • <em> - para texto enfatizado/itálico
  • <u> - para texto sublinhado
  • <mark> - para texto destacado/marcado
  • <small> - para texto menor
  • <del> - para texto riscado/deletado
  • <ins> - para texto inserido/sublinhado

Divisões e Seções

<div> - Este texto está dentro de uma div (divisão). É um contêiner genérico usado para agrupar elementos.

Elemento de Código

Para mostrar código, use a tag <code>:

console.log("Hello, World!");

Elemento Pré-formatado

O elemento <pre> preserva espaços e quebras de linha:

    function exemploHTML() {
        console.log("Texto pré-formatado");
        console.log("Mantém    espaços    e");
        console.log("quebras de linha");
    }
